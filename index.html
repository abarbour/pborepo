PBO Repo

A repository for metadata, flatfiles, etc. regarding Plate Boundary Observatory (PBO) stations and data products.

What is this project?

This is a self-maintained repository for any type of meta information for Plate Boundary Observatory (PBO) stations I find useful for my own research. Feel free to contact me (@abarbour) should you wish to see other data included. More information may be found on the PBO website.

This website is organized by the section headings, with links to the @github "blobs" for each dataset; from there you can access the raw data should you need to. The section at the end of the page (NEWS) lists brief descriptions of any changes or significant updates.

PBO borehole strainmeters (BSM)

bsm_azimuths

  • Orientations of the individual BSM channels Note: if Ch0==0 then it is an assumed value (and likely not correct)

bsm_coords

  • Coordinates of BSM stations, from the latest listing.

bsm_gaps

  • By station: capacitance-bridge gap separations, install/lin. dates; these are generally used for strain data linearization.

bsm_igppGeod

  • Geodetics from my office to the PBO BSM sites.

bsm_regions

  • Regional classifications; these are found in the long (16-character) station names.

PBO borehole network

pbo_stations_raw

  • This is a raw listing of PBO stations, from NCEDC/IRIS-DMC; the values are comma-separated and some fields have character strings in quotations.

Note: the following are derived from the raw listing:

pbo_stations_instrumentcount

  • Counts of the various instrument types in the network.

pbo_stations_accel

  • List of stations having strong motion accelerometers.

pbo_stations_bsm

  • List of stations having borehole strainmeters.

pbo_stations_lsm

  • List of stations having laser strainmeters.

pbo_stations_pp

  • List of stations having pore pressure sensors.

pbo_stations_seis

  • List of stations having seismometers (geophones).
